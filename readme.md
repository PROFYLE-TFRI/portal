
# PROFYLE Portal

Welcome to the profyle portal.


## Installation

```sh
git clone https://github.com/PROFYLE-TFRI/portal
cd portal
npm install
cd client
npm install
```

## Running

```sh
npm start
# or `npm start:watch` to restart the server on every file change
```

And on a separate terminal:

```sh
cd client
npm start
```
